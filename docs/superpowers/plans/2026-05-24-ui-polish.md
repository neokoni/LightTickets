# UI Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Three UI polish changes — sidebar alignment via flex, navbar cleanup, and 关闭→无效 rename with new tab.

**Architecture:** Frontend-only changes in Vue SFCs. No backend/type changes. Sidebar alignment replaces grid with flex layout for natural top-edge alignment. Status rename is label+icon only — the `closed` enum value stays.

**Tech Stack:** Vue 3.5, TailwindCSS 4, @iconify/vue (lucide icons)
